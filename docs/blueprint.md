# **App Name**: Civitas

## Core Features:

- Issue Reporting: Allow users to report civic issues (potholes, street lights, garbage, broken signs, fallen trees, and graffiti) with detailed descriptions.
- Location Pinning: Enable users to pin the exact location of the issue on a map.
- Issue Categorization: Automatically categorize reported issues using details and location as a tool for issue type LLM.
- Dashboard: Display a personal dashboard with the list of reported issues and their current status (resolved/unresolved).
- Confirmation Message: Show a confirmation message upon issue submission: 'Your request has been submitted.'
- Light/Dark Theme Toggle: Implement a toggle for switching between light and dark themes.
- Language selection: The users should be able to select the app language to their preference.

## Style Guidelines:

- Primary color: White, clean and modern.
- Secondary color: Purple, for accents and highlights.
- Font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, balancing modernity and readability.
- Use clear and recognizable icons for issue categories (potholes, street lights, etc.) to ensure ease of use.
- Symmetrical layout with evenly aligned boxes for issue details, location information, and reporting actions to maintain a clean and organized interface.
- Subtle transitions and animations for a smooth user experience, such as the confirmation message appearing after issue submission.
